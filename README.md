## Sobre o React CRUD

<p>Esse APP foi desenvolvido especialmente para o processo seletivo Ligue junto a aplicação de backend Node/Adonis.</p>

<p>Foi feito o deploy na Heroku e está disponível no link: https://react-dev-crud.herokuapp.com/</p>

<p>Esse APP consome uma API REST via Axios, listando assim os desenvolvedores cadastrados no banco de dados.</p>
